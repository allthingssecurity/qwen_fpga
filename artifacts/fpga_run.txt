weight tile to the engine's OCL AXI-Lite BRAM over PCIe, pulsed start, polled done
(status bit0 = 0x1), and read back 64 int16 results. All 64 match the golden
dequant matvec the model check uses. This is the int8 matvec datapath (the block
every Qwen layer is built from) executing on the chip, not a bandwidth benchmark
and not the CPU model. The full 24-layer model on the chip is still ahead.
